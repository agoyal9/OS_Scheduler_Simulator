\# OS Process Scheduler \& Deadlock Simulator



\## Overview

This project simulates CPU scheduling algorithms (FCFS, SJF, Round Robin) and deadlock detection.



\## Features

\- FCFS, SJF, Round Robin scheduling

\- Deadlock detection with resource allocation

\- Input-driven using CSV files

\- Text-based visualization



\## Usage

1\. Place CSV files in the project folder.

2\. Run: 

python main.py

3\. Output will display scheduling results and deadlocked processes.
